## Lab Questions

### In plain English describe the function and use of a multiplexer.
- the multiplexer takes a number of inputs and using a selecter chooses one of the inputs, whichever is true to produce the desired output. 
### In plain English describe the function and use of a demultiplexer.
- The demultiplexer takes one input and assigns to whichever bitwise output is true in comparision with the input.
### What other uses might these circuits have? (Think Shannon’s)
- the most common application is bus arbitration within CPUs or other similar circuits. In using Shannon's information theory we could optize our data transmission or channel capacity. In conclusion they act as tools for enhancing efficiency.
